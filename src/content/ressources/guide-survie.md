## CONSEILS DE PRONONCIATION

### Polonais
- **cz** = "tch" (comme dans tchèque)
- **sz** = "ch" (comme dans chat)
- **ł** = "w" (comme dans water en anglais)
- **ń** = "gn" (comme dans montagne)
- **ę** = nasalisé (comme "in" français)

### Tchèque
- **č** = "tch"
- **š** = "ch"
- **ř** = "rj" (son unique, difficile)
- **ě** = "yé"
- L'accent est toujours sur la première syllabe

### Hongrois
- **sz** = "s"
- **s** = "ch"
- **gy** = "dy"
- **ny** = "gn"
- **ö, ü** = sons arrondis (comme en allemand)
- L'accent est toujours sur la première syllabe

### Roumain
- **ă** = "e" muet (comme dans "le")
- **â, î** = "uh" (son entre "i" et "e")
- **ț** = "ts"
- **ș** = "ch"
- Accent tonique variable

### Croate/Serbe
- **č** = "tch"
- **š** = "ch"
- **ž** = "j" (comme dans "je")
- **ć** = "tch" plus doux
- **dž** = "dj"
- **lj** = "ly"
- **nj** = "gn"

### Bulgare
- Utilise l'alphabet cyrillique
- Prononciation assez phonétique
- **ъ** = son entre "e" et "a"
- **х** = "kh" guttural
- **ж** = "j" (comme dans "je")
- **ш** = "ch"
- **ч** = "tch"
