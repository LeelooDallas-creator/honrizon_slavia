## DOCUMENTS & ADMINISTRATIF

### Obligatoires

- [ ] Passeport OU Carte d'identité (validité > 6 mois après retour)
- [ ] Photocopies des documents (séparées des originaux)
- [ ] Photos des documents dans le cloud (Drive, Dropbox)
- [ ] Carte Européenne d'Assurance Maladie (CEAM) - gratuite sur Ameli.fr
- [ ] Assurance voyage (rapatriement, frais médicaux, annulation)
- [ ] Permis de conduire (si location voiture)
- [ ] Permis de conduire international (pour certains pays hors UE)

### Réservations

- [ ] Billets d'avion/train (version papier + numérique)
- [ ] Confirmations d'hébergement
- [ ] Réservations de visites/activités payantes
- [ ] Location voiture (si applicable)

### Finances

- [ ] Carte bancaire principale
- [ ] Carte bancaire de secours (différente banque)
- [ ] Carte sans frais à l'étranger (Revolut, Wise, N26)
- [ ] 100-200€ en espèces (à changer sur place)
- [ ] Numéros pour faire opposition (notés séparément)

### Contacts d'urgence

- [ ] Numéro ambassade/consulat français
- [ ] Contacts famille/amis en France
- [ ] Contacts hébergements
- [ ] Numéro assurance voyage (disponible 24h/24)

## BAGAGES & VÊTEMENTS

### Sac/Valise

- [ ] Sac à dos (40-60L) OU valise cabine
- [ ] Petit sac à dos journée (anti-vol recommandé)
- [ ] Cadenas pour casiers auberges
- [ ] Pochette tour de cou/ceinture pour documents

### Vêtements (selon saison)

#### Printemps/Automne (mai-juin, sept-oct)

- [ ] 3-4 t-shirts/tops
- [ ] 1-2 chemises/blouses
- [ ] 1 pull/sweat
- [ ] 1 veste légère coupe-vent
- [ ] 1 pantalon confortable
- [ ] 1 jean/pantalon chaud
- [ ] Sous-vêtements (5-7 jours)
- [ ] Chaussettes (dont 1 paire chaude)
- [ ] Chaussures de marche confortables
- [ ] Sandales/tongs
- [ ] Écharpe/foulard

#### Été (juil-août)

- [ ] Shorts/jupes
- [ ] Maillot de bain
- [ ] Chapeau/casquette
- [ ] Lunettes de soleil
- [ ] Tongs/sandales

#### Hiver (nov-mars)

- [ ] Manteau chaud/doudoune
- [ ] Gants, bonnet, écharpe
- [ ] Pulls chauds (2-3)
- [ ] Thermiques (haut + bas)
- [ ] Bottes imperméables
- [ ] Chaussettes chaudes

### Accessoires vestimentaires

- [ ] Tenue correcte pour églises/monastères (épaules + genoux couverts)
- [ ] Tenue un peu habillée (restaurants, soirées)
- [ ] Vêtements de pluie/poncho
- [ ] Sac étanche pour protéger vêtements

## AVANT DE PARTIR

### À la maison

- [ ] Arrêter courrier/journaux
- [ ] Vider frigo des produits périssables
- [ ] Débrancher appareils électriques
- [ ] Vider poubelles
- [ ] Arroser plantes (ou prévoir système)
- [ ] Confier clés à voisin/ami de confiance
- [ ] Prévenir banque du voyage (éviter blocage carte)
- [ ] Activer itinérance données mobile (si besoin)
- [ ] Scanner/photographier tous documents importants

### Check final 24h avant

- [ ] Vérifier conditions météo destination
- [ ] Consulter site France Diplomatie (alertes sécurité)
- [ ] Imprimer/télécharger billets transport
- [ ] Vérifier enregistrement en ligne (si applicable)
- [ ] Recharger tous appareils électroniques
- [ ] Changer quelques euros en devise locale (ou prévoir ATM à l'arrivée)
- [ ] Vérifier poids bagages
- [ ] Prévenir proche de l'arrivée

## CONSEILS FINAUX

**Règle d'or :** Préparez tout, puis enlevez 20% (on surcharge toujours!)

**Test du portage :** Marchez 15 min avec votre sac. S'il est trop lourd, allégez.

**Lavage sur place :** Comptez laver vêtements tous les 4-5 jours (auberges, laveries, lavabos)

**Pharmacies locales :** Tout se trouve sur place, pas besoin de tout emporter

**L'essentiel :** Documents + argent + téléphone = reste est remplaçable!
